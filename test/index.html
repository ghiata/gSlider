jQuery gSlider plugin

gSlider is a versatile, cross-browser, jQuery image slider, that supports cropping and upscaling.

Usage

To use gSlider, you have to include gSlider.js and gSlider.css, add the images inside a .gSlider container and initialize the plugin: 

                
                    <link rel="stylesheet" href="/gSlider.css">
                    <script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
                    <script src="../gSlider.js"></script>
                    <script src="js/index.js"></script>

                    <div class="gSlider">
                        <img src="images/1.jpg" />
                        <img src="images/2.jpg" />
                        <img src="images/3.jpg" />
                        <img src="images/4.jpg" />
                        <img src="images/5.jpg" />
                    </div>

                    <script>
                        $(document).ready(function() {
                            $('.gSlider').gSlider();
                        });
                    </script>

Features

  • Initialization with custom slide interval (default is 5000ms): 
    $('.gSlider').gSlider({interval: 5000});
  • Sliding is paused when hovering the slider. This is achieved with javascript, by clearing the interval on mouse hover
  • Images are centered and cropped. This is achieved with CSS3 by wrapping the img tags with divs that have the background-image centered and background-size: cover;
  • Upscales pictures in case they are to small (see bottom of page)
